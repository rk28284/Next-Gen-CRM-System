
  <h1 align="center">Next Gen CMS - Full CRUD Operations</h1>

  <h2>Project Overview</h2>
  <p>
    This Next Gen CMS is a modern platform supporting full Create, Read, Update, and Delete (CRUD) operations.
  </p>

  <h2>Features Include</h2>
  <ul>
    <li>Create new content and manage data easily through the intuitive React front-end.</li>
    <li>Read and display all stored content dynamically using React components.</li>
    <li>Update existing content with real-time form inputs and API integration.</li>
    <li>Delete unwanted content securely with confirmation prompts.</li>
  </ul>

  <h2>Technology Stack</h2>
  <ul>
    <li><strong>Frontend:</strong> React, TailwindCSS, JavaScript, HTML, CSS</li>
    <li><strong>Backend:</strong> Node.js, Express, Mongoose (MongoDB)</li>
  </ul>

  <h2>CRUD API Endpoints</h2>
  <ul>
    <li><code>POST /api/lead</code> - Create new content</li>
    <li><code>GET /api/lead</code> - Read/List all content</li>
    <li><code>GET /api/lead/:id</code> - Read single content by ID</li>
    <li><code>PUT /api/lead/:id</code> - Update content by ID</li>
    <li><code>DELETE /api/lead/:id</code> - Delete content by ID</li>
  </ul>

  <h2>How CRUD Works</h2>
  <ol>
    <li><strong>Create:</strong> Users submit forms that send <code>POST</code> requests, adding new data to the database.</li>
    <li><strong>Read:</strong> Data is fetched via <code>GET</code> requests and rendered in frontend components dynamically.</li>
    <li><strong>Update:</strong> Edited content is sent with <code>PUT</code> requests to modify existing database records.</li>
    <li><strong>Delete:</strong> <code>DELETE</code> requests remove records after user confirmation.</li>
  </ol>

  <h2>Summary</h2>
  <p>
    This CMS project fully supports CRUD operations both on the UI and API levels, ensuring seamless management of website content.
  </p>
